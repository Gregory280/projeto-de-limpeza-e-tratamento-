# Projeto de Limpeza e Tratamento de Valores Ausentes
Este projeto tem como objetivo realizar um processo completo de **limpeza de dados**, **tratamento de valores ausentes** e **detecção e tratamento de outliers** para preparar um dataset para análises exploratórias e modelagem.

### Objetivo
O objetivo principal deste projeto é transformar um dataset bruto em um conjunto de dados mais limpo, consistente e enriquecido, pronto para análises avançadas.

### Resultados
Após o processo de limpeza e enriquecimento:
* O dataset ficou livre de valores ausentes
* Outliers de colunas específicas foram substituidos
* Novos atributos que auxiliam analises posteriores
